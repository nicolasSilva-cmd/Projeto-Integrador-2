#  **Projeto Integrador II**  

<p> Projeto utilizando framework Spring, que deverá resolver o problema da gestão de livros e de alunos, como controle de devoluções e de livros disponíveis. </p>
<br>

<p> Devem ser utilizados: </p> 

<ol>

  <li>Dentre outros...</li>
</ol>
